## Installation

```bash
$ yarn 

# generate prisma client
$ npx prisma generate
``` 

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```

## Deploy

```bash
# build first
$ yarn build

# deploy to vercel
$ vercel deploy --prod
```
